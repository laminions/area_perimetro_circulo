# programa 1: area_perimetro_circulo
programa en python para calcular el area y el perimetro de un circulo dado el valor de su radio

## Analisis

### Variables de entrada
- r:radio del circulo

### procesamiento
- a: area del circulo
- p: perimetro de circulo 

$a = \pi*r^2$
$p = 2*\pi*r$
